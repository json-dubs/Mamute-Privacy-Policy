# Mamute Privacy Policy (Public Repo)

This folder is a publish-ready package for a standalone public GitHub repository
that hosts the privacy policy URL required by Google Play.

## Recommended setup

1. Create a new public repo, for example `mamute-privacy-policy`.
2. Copy these files into that repo root:
   - `index.html`
   - `PRIVACY_POLICY.md`
3. In GitHub, open:
   - `Settings` -> `Pages`
   - `Build and deployment` -> `Source: Deploy from a branch`
   - `Branch: main` and `/ (root)`
4. Use the live URL in Play Console for both apps:
   - `https://<your-github-username>.github.io/mamute-privacy-policy/`

## Apps covered by this policy

- `com.jsondubs.mamutestudent`
- `com.jsondubs.mamuteinstructor`
